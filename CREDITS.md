# credits
This file lists all content sources and contributors to the project.

## content sources
- [Flutter](https://flutter.dev/) - the framework used to build the app
- [Supabase](https://supabase.com/) - the backend provider used for database, auth and analytics
- [Cloudflare](https://www.cloudflare.com/) - the data storage provider used for storing videos, profile images, thumbnails and subtitles
- [dicebear](https://dicebear.com/) - the source for the default profile images
- [Flutter packages](https://pub.dev/) - the source for all flutter packages used in the project, look at the pubspec.yaml file for the exact packages used

- [Kurzgesagt - In a Nutshell](https://www.youtube.com/@Kurzgesagt) - the source for some placeholder videos used in the app. At [Kurzgesagt's FAQ](https://kurzgesagt.org/faq/) they state that you can use the videos if you credit them and link to their channel and only use them if you stream them via youtube, which i did.
